# 🔹 1. Acceso a formularios con DOM

``` html
<form name="formulario" id="formulario">
  <input type="text" name="elemento" id="elemento" />
</form>
```

``` javascript
// Obtener el formulario por su id
var formularioPrincipal = document.getElementById("formulario");

// Obtener un elemento concreto del formulario
var primerElemento = document.getElementById("elemento");
```

------------------------------------------------------------------------

# 🔹 2. Propiedades importantes de los elementos

``` javascript
var elemento = document.getElementById("elemento");

elemento.type;   // Tipo del elemento (text, checkbox, select-one, etc.)
elemento.form;   // Referencia directa al formulario al que pertenece
elemento.name;   // Nombre del elemento (solo lectura)
elemento.value;  // Valor del campo (se puede leer y modificar)
```

------------------------------------------------------------------------

# 🔹 3. Eventos comunes en formularios

``` html
<input type="text" onchange="alert('Valor cambiado')" />
<input type="text" onfocus="alert('Campo seleccionado')" />
<input type="text" onblur="alert('Campo perdió el foco')" />
<input type="button" onclick="alert('Botón pulsado')" />
```

// onclick → al hacer clic\
// onchange → cuando cambia el valor\
// onfocus → cuando recibe el foco\
// onblur → cuando pierde el foco

# 🔹 4. Obtener valor de input text y textarea

``` html
<input type="text" id="texto" />

```

``` javascript
// Obtener valor del input
var valorTexto = document.getElementById("texto").value;

// Obtener valor del textarea
var valorParrafo = document.getElementById("parrafo").value;
```

# 🔹 5. Radiobutton

``` html
<input type="radio" value="si" name="pregunta" id="pregunta_si"/> SI
<input type="radio" value="no" name="pregunta" id="pregunta_no"/> NO
<input type="radio" value="nsnc" name="pregunta" id="pregunta_nsnc"/> NS/NC
```

``` javascript
// Obtener todos los radios del mismo grupo (mismo name)
var elementos = document.getElementsByName("pregunta");

for (var i = 0; i < elementos.length; i++) {

    // checked es true solo en el radio seleccionado
    alert("Elemento: " + elementos[i].value +
          "\nSeleccionado: " + elementos[i].checked);
}
```

# 🔹 6. Checkbox

``` html
<input type="checkbox" value="condiciones" id="condiciones"/>
<input type="checkbox" value="privacidad" id="privacidad"/>
```

``` javascript
// Comprobar cada checkbox individualmente
var elemento = document.getElementById("condiciones");

alert("Elemento: " + elemento.value +
      "\nSeleccionado: " + elemento.checked);

elemento = document.getElementById("privacidad");

alert("Elemento: " + elemento.value +
      "\nSeleccionado: " + elemento.checked);
```

# 🔹 7. Select (lista desplegable)

``` html
<select id="opciones">
  <option value="1">Primer valor</option>
  <option value="2">Segundo valor</option>
  <option value="3">Tercer valor</option>
  <option value="4">Cuarto valor</option>
</select>
```

``` javascript
// Obtener la referencia a la lista
var lista = document.getElementById("opciones");

// Obtener índice seleccionado (empieza en 0)
var indiceSeleccionado = lista.selectedIndex;

// Obtener opción seleccionada usando el array options
var opcionSeleccionada = lista.options[indiceSeleccionado];

// Obtener texto visible
var textoSeleccionado = opcionSeleccionada.text;

// Obtener valor real definido en el atributo value
var valorSeleccionado = opcionSeleccionada.value;

alert("Opción: " + textoSeleccionado +
      "\nValor: " + valorSeleccionado);
```

## 🔹 Versión abreviada

``` javascript
var lista = document.getElementById("opciones");

var valorSeleccionado =
    lista.options[lista.selectedIndex].value;

var textoSeleccionado =
    lista.options[lista.selectedIndex].text;
```

📌 IMPORTANTE: - selectedIndex → posición en el array (empieza en 0). -
value → valor real definido en la opción. - No son lo mismo.

# 🧾 Resumen Final

En formularios se trabaja principalmente con:

-   document.getElementById()
-   Propiedad .value
-   Propiedad .checked
-   Propiedad .selectedIndex
-   Array options
-   Eventos: onclick, onchange, onfocus, onblur

Permiten validar, leer y manipular datos introducidos por el usuario.
