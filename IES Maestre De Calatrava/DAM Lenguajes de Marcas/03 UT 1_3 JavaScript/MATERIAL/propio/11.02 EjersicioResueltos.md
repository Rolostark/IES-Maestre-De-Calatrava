## Bloque 1: Interacción con el DOM y Estilos (Ejercicios 1 - 9)

### Ejercicio 1: Cambiar contenido con innerHTML

Completa el documento que incluye una etiqueta de botón para cambiar el texto de un párrafo.

HTML

```html
<!DOCTYPE html>
<html>
<body>
  <p id="demo">Texto original</p>
  <button type="button" onclick="document.getElementById('demo').innerHTML = 'Bienvenido a Java Script!'">
    Haz clic aquí
  </button>
</body>
</html>
```

**Explicación:** Se utiliza el método `getElementById` para encontrar el elemento y la propiedad `innerHTML` para modificar su contenido.

### Ejercicio 2: Cambiar atributo src (La Bombilla)

Documento donde JavaScript cambia el valor del atributo de una imagen al hacer clic.

HTML

```html
<!DOCTYPE html>
<html>
<body>
  <img id="myImage" onclick="changeImage()" src="pic_bulboff.gif" width="100" height="180">
  <script>
    function changeImage() {
      var image = document.getElementById('myImage');
      if (image.src.match("bulbon")) {
        image.src = "pic_bulboff.gif";
      } else {
        image.src = "pic_bulbon.gif";
      }
    }
  </script>
</body>
</html>
```

**Explicación:** JavaScript puede cambiar los atributos HTML, en este caso modificando el `src` de una etiqueta `<img>`.

## Bloque 3: Estructuras de Datos y Objetos (Ejercicios 21 - 27)

### Ejercicio 21: Matriz de Meses

Crear un array y acceder a posiciones específicas.

HTML

```html
<!DOCTYPE html>
<html>
<body>
  <p id="todos"></p>
  <p id="junio"></p>
  <script>
    var meses = ["Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio", 
                 "Julio", "Agosto", "Septiembre", "Octubre", "Noviembre", "Diciembre"];
    document.getElementById("todos").innerHTML = meses;
    document.getElementById("junio").innerHTML = "Posición 5 (Junio): " + meses[5];
  </script>
</body>
</html>
```

**Explicación:** Las matrices se escriben con corchetes y sus índices empiezan en la posición 0.

## Bloque 4: Lógica y Bucles (Ejercicios 28 - 41)

### Ejercicio 29: Validar si es número (isNaN)

Comprobar si el dato de un `input` es numérico.

HTML

```html
<!DOCTYPE html>
<html>
<head>
  <script>
    function esNumero() {
      var x = document.getElementById("numb").value;
      if (isNaN(x)) {
        alert("No es un número");
      } else {
        alert("Es un número");
      }
    }
  </script>
</head>
<body>
  <input id="numb">
  <button type="button" onclick="esNumero()">¿Es número?</button>
</body>
</html>
```

### Ejercicio 32: Saludos condicionales

Estructura `if / else if` basada en la hora actual.

HTML

```html
<!DOCTYPE html>
<html>
<body>
  <p id="demo"></p>
  <script>
    var hora = new Date().getHours();
    var greeting;
    if (hora < 13) {
      greeting = "¡Buen día!";
    } else if (hora < 18) {
      greeting = "Buenas tardes";
    } else {
      greeting = "Buenas noches";
    }
    document.getElementById("demo").innerHTML = greeting;
  </script>
</body>
</html>
```

### Ejercicio 35 y 36: Bucle for

Generar listas y recorrer arrays.

HTML

```html
<!DOCTYPE html>
<html>
<body>
  <p id="lista"></p>
  <p id="coches"></p>
  <script>
    // Ejercicio 35: Números del 0 al 9
    var i, texto = "";
    for (i = 0; i < 10; i++) {
      texto += i + "<br>";
    }
    document.getElementById("lista").innerHTML = texto;

    // Ejercicio 36: Recorrer array
    var cars = ["BMW", "Volvo", "Saab", "Ford"];
    var res = "";
    for (i = 0; i < cars.length; i++) {
      res += cars[i] + "<br>";
    }
    document.getElementById("coches").innerHTML = res;
  </script>
</body>
</html>
```

### Ejercicio 41: Bucle while (Suma interactiva)

Suma números hasta que el usuario introduzca un 0.

HTML

```html
<!DOCTYPE html>
<html>
<body>
  <script>
    var suma = 0, n = -1;
    while (n !== 0) {
      n = parseInt(prompt("Introduce un número (0 para terminar):", "0"));
      if (!isNaN(n)) {
        suma += n;
      }
    }
    alert("La suma total es: " + suma);
  </script>
</body>
</html>
```
